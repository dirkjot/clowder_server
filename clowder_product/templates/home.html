You have a script you need to run monthly.

Just add ~1 line of code to the bottom of your script.


Python - Node - Ruby - PHP

import clowder
import datetime

clowder.Watch(
    name='Monthly Job',
    schedule=datetime.timedelta(days=31),
).ok()

You want an alert when CPU usage goes over 60%.

Just add ~1 line of code to the bottom of your script.


Python - Node - Ruby - PHP

import clowder
import psutil

cpu_percentage = psutil.cpu_percent()

clowder.Watch(
    name='CPU Percentage',
    alert=lambda x: x >= 60
).submit(cpu_percentage)
